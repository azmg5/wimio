# Single Page Application with Laravel 5.3 and Vue.js 2.1.x

### Frameworks and Libraries:

- Laravel 5.3
- Vue.js 2.1.x
- Vue Router
- axios
- Google Maps

### Provides:

- CRUD
- Single Page Application
- Pagination
- Sorting by columns
- Advance filter
- Filter relations
- Google Maps

### Installation
`git clone https://github.com/azmg5/wimo.git`

`cd wimo`

`composer install`
